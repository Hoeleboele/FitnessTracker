A static web app that can track my fitness journey in a way that looks like a rpg.
I want the app to keep track of my current lifting weight in kg and amount of reps.
I need to be able to add a workout that would contain multiple practices which i can choose from a drop down showing the names with a name, weight kg, amount of reps.
I should be able to add new practices that i can see when making am workout
I should be able to start a workout. When i start a workout i should see a list of all exercises in the workout.
I should be able to look back on the workouts i did so i can see my progress in weight and reps.
I should be able to select workouts and delete or edit them.
When preforming an workout i should be able to start a timer that keeps track of my rest time.
When preforming an workout i should be able to open an exercise to see the weight and reps.
When creating a workout i should be able to add an rest time for that workout.
When i am done with an workout i should be able to lower or change the weight and rep amount for the next time.
The app should be made with html, javascript and css.
The app will be hosted on github pages so a deployment yml is needed
The app should have the posibility to play music through my spotify
